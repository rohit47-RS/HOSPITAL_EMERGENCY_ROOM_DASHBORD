# HOSPITAL_EMERGENCY_ROOM_DASHBORD
 A Power BI report analyzing ER patient data across gender, race, age, department referrals, and admission status. Features KPI cards (patient count, wait time, satisfaction score, referrals), Year/Month/Department slicers, combo/pie/bar charts, a tree map, and a pivot table, built on a two-table (hospital data + Calendar) star-schema model.
